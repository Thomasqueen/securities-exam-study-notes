## 一、使用边界

这份资料用于补充官方大纲和统编教材，不替代教材。

使用原则：

- 官方大纲和教材负责“考什么”。
- B 站视频负责“老师怎么讲、题目怎么拆、易错点怎么记”。
- 所谓“真题、押题、重复率很高”等说法只作风险提示，不作为复习依据。
- 看视频时不追求全部看完，优先看自己错题对应章节。

## 三、老师讲题常见题型总结

### 1. 概念识别题

形式：

- “下列关于 X 的说法，正确/错误的是”
- “下列属于/不属于 X 的是”

高频考点：

- 直接融资与间接融资
- 货币市场与资本市场
- 股票与债券
- 普通股与优先股
- 公募基金与私募基金
- 期货与期权
- 自营、经纪、资管、投顾、承销保荐

做题方法：

- 先定义，再排除。
- 看到“不属于”，先圈题干。
- 警惕把一个概念的特征套到另一个概念上。

### 2. 范围列举题

形式：

- “下列属于证券公司业务范围的有”
- “下列属于债券投资风险的有”
- “下列属于内幕信息知情人的有”

高频考点：

- 证券公司业务种类
- 证券服务机构类别
- 机构投资者分类
- 基金类型
- 债券类型
- 金融衍生品分类
- 证券违法违规行为类型

做题方法：

- 多选题不要凭感觉，逐项判断。
- 题目问“业务范围”，就不要选“监管机构职责”。
- 题目问“法律责任”，就区分民事、行政、刑事、自律。

### 3. 易混对比题

形式：

- “关于 A 与 B 的区别，下列说法正确的是”
- “下列关于 X 和 Y 的说法，错误的是”

高频对比：

- 注册制 vs 核准制
- 股票 vs 债券
- 公司债 vs 企业债
- 可转债 vs 可交换债
- ETF vs LOF
- 公募 vs 私募
- 期货 vs 远期
- 期货 vs 期权
- 投顾建议 vs 代客操作
- 行政处罚 vs 自律处分

做题方法：

- 把区别背成“主体、对象、权利义务、交易场所、风险特征”。
- 一题只抓一个核心区别，不要被长选项带跑。

### 4. 场景合规题

形式：

- 给出营业部员工、客户、产品、交易行为，问是否违规。
- 给出券商业务场景，问违反哪项规定。

高频场景：

- 从业人员代客操作。
- 承诺收益或赔偿损失。
- 私下接受客户委托。
- 未履行适当性义务。
- 未充分揭示风险。
- 泄露客户信息。
- 利用内幕信息或未公开信息交易。
- 研究报告利益冲突未披露。
- 资管产品承诺保本保收益。

做题方法：

- 第一步圈主体：证券公司、员工、客户、发行人、中介机构。
- 第二步圈行为：开户、营销、交易、咨询、资管、融资融券、披露。
- 第三步找红线：代客、承诺、泄露、内幕、误导、不匹配。

### 5. 数字期限题

形式：

- “下列关于期限/比例/时间的说法正确的是”
- “某行为发生后，应在多久内披露/保存/处理”

高频方向：

- 成绩有效期。
- 信息披露期限。
- 客户资料和交易记录保存期限。
- 反洗钱资料保存。
- 持股比例和权益变动披露节点。
- 短线交易期限。
- 融资融券保证金、维持担保比例。
- 基金申购赎回、估值、披露。

做题方法：

- 不把数字零散背，按“披露类、保存类、交易类、资格类、风控类”归类。
- 考前单独做数字表。

### 6. 综合材料题

形式：

- 一段材料中混合 2-4 个知识点。
- 常见于法规科业务场景和金融科产品/估值/风险场景。

做题方法：

- 先拆材料，不急着看选项。
- 金融科看“产品 + 风险 + 收益 + 市场”。
- 法规科看“主体 + 行为 + 义务 + 责任”。

## 四、看视频后补充到背诵清单的高频点

### 金融科补丁

#### 1. 注册制专题

必须背：

- 注册制以信息披露为核心。
- 注册制下发行人是信息披露第一责任人。
- 中介机构应勤勉尽责。
- 注册制不等于监管不审核、不监管、不担责。
- 全面注册制强调把选择权交给市场，同时强化事中事后监管。

常见考法：

- 把注册制说成“只要提交材料就能发行”，错误。
- 把注册制说成“中介机构无需担责”，错误。

#### 2. 多层次资本市场专题

必须背：

- 主板：成熟企业。
- 科创板：硬科技、科技创新企业。
- 创业板：成长型创新创业企业。
- 北交所：创新型中小企业。
- 新三板：全国中小企业股份转让系统。
- 区域性股权市场：服务区域内中小微企业。

常见考法：

- 板块定位互换。
- 北交所和新三板关系混淆。

#### 3. 股票交易专题

必须背：

- 竞价交易遵循价格优先、时间优先。
- 集合竞价在一定时间内集中撮合。
- 连续竞价逐笔连续撮合。
- 涨跌幅限制属于市场稳定机制。
- 大宗交易适用于达到一定规模的证券交易。

常见考法：

- 把集合竞价说成逐笔成交，错误。
- 把做市商交易和竞价交易混同，错误。

#### 4. 债券估值专题

必须背：

- 债券价格通常与市场利率反向变动。
- 到期收益率是使未来现金流现值等于债券当前价格的贴现率。
- 信用风险越高，投资者要求收益率越高。
- 债券期限越长，通常利率风险越大。

常见考法：

- 利率上升，固定利率债券价格通常下降。
- 信用评级下降，债券价格通常承压。

#### 5. 基金专题

必须背：

- 基金管理人负责投资运作。
- 基金托管人负责资产保管和监督。
- 基金份额持有人享有收益并承担风险。
- 认购发生在募集期，申购发生在基金成立后。
- 公募公开募集，私募非公开面向合格投资者募集。

常见考法：

- 把管理人和托管人职责互换。
- 把认购和申购时间混淆。

#### 6. 衍生品专题

必须背：

- 期货是标准化合约，通常场内集中交易。
- 远期通常非标准化，常见于场外交易。
- 期权买方有权利无义务，卖方有履约义务。
- 期货实行保证金和每日无负债结算。
- 衍生品具有杠杆性，风险放大。

常见考法：

- 把期权买方说成必须行权，错误。
- 把远期和期货的标准化程度混淆。

#### 7. 风险管理专题

必须背：

- 市场风险：价格、利率、汇率等市场变量变化导致损失。
- 信用风险：交易对手或债务人违约导致损失。
- 流动性风险：无法及时以合理成本变现或融资。
- 操作风险：人员、流程、系统或外部事件导致损失。
- 声誉风险：负面评价损害机构品牌和信任。

常见考法：

- 给一个风险事件，判断风险类型。
- 把操作风险和市场风险混淆。

### 法规科补丁

#### 1. 证券法主线

必须背：

- 三公原则：公开、公平、公正。
- 信息披露：真实、准确、完整、及时、公平。
- 内幕信息：未公开 + 重大。
- 禁止内幕交易、操纵市场、虚假陈述、欺诈客户。
- 投资者保护贯穿发行、交易、信息披露和纠纷处理。

常见考法：

- 给出行为，判断属于内幕交易、操纵市场还是虚假陈述。

#### 2. 公司法专题

必须背：

- 分公司不具有法人资格，子公司具有法人资格。
- 有限责任公司股东依法享有知情权。
- 股份有限公司可采取发起设立或募集设立。
- 董事、高级管理人员负有忠实义务和勤勉义务。
- 控股股东、实际控制人不得滥用控制地位损害公司或股东利益。

常见考法：

- 分公司和子公司法律地位。
- 董监高义务。
- 股份转让和回购规则。

#### 3. 适当性管理专题

必须背：

- 了解客户。
- 了解产品。
- 风险揭示。
- 适当匹配。
- 普通投资者享有特别保护。
- 不得向风险承受能力不匹配的客户主动推介高风险产品。

常见考法：

- 客户风险承受能力低，却被推荐高风险产品。
- 销售人员只讲收益不讲风险。

#### 4. 经纪业务专题

必须背：

- 经纪业务是接受客户委托代理买卖证券。
- 不得接受全权委托。
- 不得代客操作。
- 不得承诺收益。
- 不得诱导频繁交易。
- 不得擅自变更客户指令。
- 应落实账户实名制、反洗钱和投资者适当性。

常见考法：

- 客户把账户密码交给员工，员工替客户交易，违规。

#### 5. 投资咨询与研究报告专题

必须背：

- 投顾可以提供建议，不得替客户作出交易决定。
- 投资建议应有合理依据。
- 不得保证收益。
- 研究报告应独立、客观、公正。
- 应披露可能影响独立性的利益冲突。
- 荐股软件本质上仍受投资咨询业务规范约束。

常见考法：

- “老师推荐后直接帮客户下单”，违规。
- 研究报告未披露利益冲突，违规。

#### 6. 资产管理专题

必须背：

- 资产管理业务是受托管理客户资产。
- 资产应独立管理、独立核算。
- 不得承诺保本保收益。
- 应履行适当性和风险揭示义务。
- 自营、经纪、资管之间应防范利益冲突。

常见考法：

- 资管销售说“公司兜底、固定收益”，违规。

#### 7. 融资融券专题

必须背：

- 融资是借钱买证券。
- 融券是借证券卖出。
- 涉及信用账户、保证金、担保物、维持担保比例。
- 证券公司应充分揭示融资融券风险。

常见考法：

- 融资和融券概念互换。
- 保证金和担保物管理。

#### 8. 违法违规行为专题

必须背：

- 欺诈发行：发行阶段信息或行为欺诈。
- 违规披露：信息披露虚假、误导、重大遗漏或不及时。
- 内幕交易：利用未公开重大信息交易。
- 操纵市场：通过不正当手段影响价格或交易量。
- 利用未公开信息交易：利用职务便利获取的非公开信息交易。
- 背信运用受托财产：违背受托义务使用客户或受托财产。

常见考法：

- 用一个材料区分“内幕交易”和“利用未公开信息交易”。
- 用一个材料区分“虚假陈述”和“操纵市场”。

#### 9. 廉洁从业与诚信管理专题

必须背：

- 不得谋取或输送不正当利益。
- 不得利用职务便利为本人或他人谋取利益。
- 诚信信息会影响执业声誉。
- 市场禁入适用于严重违法失信主体。
- 从业人员违法违规可能被纪律处分、行业惩戒、监管处罚。

常见考法：

- 礼品、回扣、利益输送场景。
- 诚信档案和市场禁入后果。
