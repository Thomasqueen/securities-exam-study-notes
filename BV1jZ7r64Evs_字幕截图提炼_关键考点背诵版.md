# 第一部分 金融市场基础知识：截图提炼背诵点

## P1 第 1 章：金融市场体系，5 分

### 截图看到的重点

- 直接融资定义。
- 直接融资方式。
- 直接融资与间接融资区别。
- 国际资金流动方式。

### 必背点

- 直接融资：资金需求方通过直接发行股票、债券等金融工具向资金供给方融资。
- 间接融资：通过银行等金融中介完成资金融通。
- 直接融资工具包括股票、债券、基金份额等证券化工具。
- 直接融资方式包括股票市场融资、债券市场融资、风险投资融资、商业信用融资、民间借贷等。
- 国际资金流动包括贸易性资金流动、套利性资金流动、保值性资金流动、投机性资金流动。

### 易错点

- 发行股票、债券属于直接融资，不属于间接融资。
- 商业银行贷款通常属于间接融资。
- 直接融资和间接融资的核心区别是“资金供求双方是否直接形成金融关系”。

### 快速背诵句

直接融资看证券，间接融资看银行；股票债券是直接，银行贷款是间接。

## P2 第 2 章：中国金融体系与多层次资本市场，10 分

### 截图看到的重点

- 商业银行主要业务。
- 货币政策目标和工具。
- 场内市场、场外市场。
- 创业板、科创板定位。
- 证券公司可在柜台市场开展的业务。

### 必背点

- 商业银行业务包括负债业务、资产业务、中间业务。
- 负债业务是商业银行资金来源，存款是重要组成部分。
- 货币政策最终目标通常包括币值稳定、经济增长、充分就业、国际收支平衡。
- 货币政策中介目标应具备可测性、可控性、相关性、抗干扰性和适应性。
- 直接信用控制包括行政命令或其他方式直接干预信用活动。
- 间接信用控制通过道义劝告、窗口指导等方式影响金融机构。
- 场内市场包括证券交易所市场。
- 场外市场包括全国股转系统、柜台市场、区域性股权市场等。
- 科创板服务符合国家战略、突破关键核心技术、市场认可度高的科技创新企业。
- 证券公司柜台市场可以开展收益凭证、报价、做市、私募产品销售与转让等业务，但不得采用集合竞价方式，法律法规另有规定的除外。

### 易错点

- 自有资金不是商业银行最主要的资金来源，存款更重要。
- 直接信用控制和间接信用控制容易混。
- 场外市场不是“没有监管的市场”。
- 区域性股权市场属于场外市场，不是证券交易所市场。

### 快速背诵句

货币政策看目标和工具；资本市场看分层；场内交易所，场外股转、柜台、区域股权。

## P3 第 3 章：证券市场主体，17 分

### 截图看到的重点

- 政府和政府机构发行债券。
- 投资者适当性。
- 证券公司资产管理业务。
- 中国证券业协会。
- 国务院证券监督管理机构职责和权限。

### 必背点

- 政府债券的发行主体包括中央政府和地方政府。
- 政府发行债券的目的包括弥补财政赤字、筹集建设资金、实施特殊政策、战争或突发事件资金需求等。
- 投资者适当性管理是通过一系列措施，让适合的投资者购买适合的产品或服务。
- 适当性管理不等于保证投资者盈利，不等于替投资者承担投资风险。
- 证券公司资产管理业务是接受客户委托，对客户资产进行经营运作。
- 资产管理计划不得公开宣传，不得向不合格投资者募集。
- 证券业协会是证券行业自律组织，会员包括证券公司会员、特别会员和联系会员。
- 国务院证券监督管理机构可查询资金账户、证券账户、银行账户等；可限制转移、隐匿违法资金和证券。

### 易错点

- 适当性管理不是“只要客户愿意买就能卖”。
- 资管计划不得向不合格投资者募集。
- 协会不是行政监管机关，是行业自律组织。
- 证监会权限常和交易所、协会职责混淆。

### 快速背诵句

主体章先分角色：发行人融资，投资者出资，中介服务，自律组织管行业，监管机构管市场。

## P4 第 4 章：股票，19 分

### 截图看到的重点

- 股票定义。
- 普通股股东权利。
- 股票交易程序：委托买卖。
- 股票价值和价格。

### 必背点

- 股票是股份有限公司签发的，证明股东所持股份和享有权益的有价证券。
- 股票实质上代表股东对公司的资产所有权。
- 普通股股东权利包括公司重大决策参与权、资产收益权、剩余财产分配权、知情权等。
- 股东大会包括年会和临时股东大会。
- 委托买卖流程包括申报原则、申报方式、申报时间。
- 证券经纪商接受委托后，通常按时间优先、客户优先原则进行申报。
- 股票理论价格是未来收益的现值。
- 股票市场价格由供求关系决定，并受预期、政策、市场情绪等影响。

### 易错点

- 股票代表股东权利，不是债权凭证。
- 普通股和优先股的权利顺序不同。
- 市场价格不一定等于内在价值。
- 委托流程中的时间优先、客户优先要和竞价撮合中的价格优先、时间优先区分。

### 快速背诵句

股票是股权凭证；普通股有表决和收益；价格看供求，价值看未来收益折现。

## P5 第 5 章：债券，19 分

### 截图看到的重点

- 债券性质。
- 金融债券发行条件。
- 企业债券品种。
- 资产证券化。
- 中小非金融企业集合票据。

### 必背点

- 债券属于有价证券，是债权债务关系的证明。
- 债券反映发行人与投资者之间的债权债务关系。
- 金融债券发行主体通常是金融机构。
- 商业银行金融债券发行条件包括良好公司治理、核心资本充足率要求、连续盈利、贷款损失准备计提充足、风险监管指标合规等。
- 企业债券品种包括专项债券、项目收益债券、可续期企业债券等。
- 资产证券化是以基础资产产生的现金流为支持发行证券。
- 资产证券化能提供多样化投资品种、降低资本要求、扩大投资规模。
- 中小非金融企业集合票据由多个中小企业集合发行，统一产品设计、统一券种冠名、统一信用增进、统一发行注册。

### 易错点

- 债券是债权凭证，股票是股权凭证。
- 公司债和企业债监管规则、发行主体和历史口径不同。
- 可转债和可交换债不同。
- 资产证券化不是普通债券融资，核心是基础资产现金流。

### 快速背诵句

债券看四要素：发行人、面值、利率、期限；债券风险看利率、信用、流动性。

## P6 第 6 章：证券投资基金，15 分

### 截图看到的重点

- 债券评级。
- 基金资产估值。
- 基金税收。
- 基金信息披露。

### 必背点

- 债券评级关注发行主体和债项偿债能力，评级机构应独立、客观、公正。
- 基金资产估值需考虑估值频率、交易价格公允性、估值方法一致性及公开性。
- 当基金投资标的交易活跃并有公允价格时，应采用市价或第三方估值。
- 基金投资者税收包括机构投资者买卖基金份额、取得分红、赎回所得等税务处理。
- 基金信息披露分为基金募集信息披露和基金运作信息披露。
- 募集信息披露包括招募说明书、基金合同、基金托管协议等。
- 运作信息披露包括净值公告、定期报告、临时报告等。

### 易错点

- 基金管理人、托管人、持有人职责容易混。
- 基金估值不是随意估值，要讲公允性、一致性、公开性。
- 募集信息披露和运作信息披露要区分。

### 快速背诵句

基金三方：持有人享权益，管理人做投资，托管人管资产和监督。

## P7 第 7 章：金融衍生品，7 分

### 截图看到的重点

- 金融衍生工具概念和特征。
- 金融互换。
- 金融期货主要交易制度。
- 金融远期合约种类。

### 必背点

- 金融衍生工具价值依赖基础金融产品。
- 衍生品包括金融远期、金融期货、金融期权、金融互换等。
- 金融互换是两个或两个以上当事人按约定条件交换现金流的合约。
- 期货交易制度包括保证金、当日无负债结算、涨跌停板、持仓限额、大户报告、强行平仓等。
- 限仓制度用于防止市场风险过度集中和防范操纵。
- 大户报告制度用于监控异常持仓风险。
- 金融远期包括股权类资产远期合约、债权类资产远期合约、远期利率协议、远期汇率协议。

### 易错点

- 期货是标准化合约，远期通常非标准化。
- 期权买方有权利无义务，卖方承担履约义务。
- 互换交换的是现金流，不是简单买卖资产。

### 快速背诵句

衍生品四兄弟：远期、期货、期权、互换；期货标准化，远期个性化。

## P8 第 8 章：金融风险管理，8 分

### 截图看到的重点

- 风险偏好和风险容忍度。
- 风险限额。
- 压力测试。

### 必背点

- 风险偏好是机构愿意承担的风险类型和水平。
- 风险容忍度是风险偏好的具体化边界。
- 风险偏好设定常采用自上而下、自下而上及二者结合方式。
- 风险限额通常包括限额设定、限额监测、限额调整、超限处理等流程。
- 压力测试是基于特定压力情景，分析市场变量剧烈变化对金融机构或资产组合的影响。

### 易错点

- 风险偏好偏战略层面，风险限额偏执行控制层面。
- 压力测试不是预测正常情景，而是检验极端情景承受能力。

### 快速背诵句

风险管理先定偏好，再设限额，日常监测，极端情景靠压力测试。

# 第二部分 证券市场基本法律法规：截图提炼背诵点

## P9 第 1 章：证券市场基本法律法规，35 分

### 截图看到的重点

- 公司合并、分立。
- 有限责任公司设立和组织机构。
- 国有独资公司组织机构。
- 禁止证券交易行为。
- 证券登记结算机构业务规则。

### 必背点

- 公司合并、分立通常需要股东会或股东大会决议，编制资产负债表和财产清单，通知或公告债权人，并由合并或分立后的主体承继债权债务。
- 公司与其持股 90% 以上的公司合并，被合并公司不需要股东会决议，但应通知其他股东，其他股东有权请求公司按合理价格收购其股权或股份。
- 有限责任公司由 1 个以上 50 个以下股东出资设立。
- 有限责任公司设立协议对公司内部有效，不得对抗善意第三人。
- 有限责任公司股东可以用货币，也可以用实物、知识产权、土地使用权、股权、债权等可估价并可依法转让的非货币财产出资。
- 国有独资公司董事会成员中应有职工代表。
- 禁止交易行为包括内幕交易、操纵市场、虚假陈述、误导性信息、欺诈客户等。
- 禁止利用虚假或不确定重大信息诱导投资者交易。
- 禁止对证券、发行人公开作出评价、预测或投资建议，并进行反向证券交易。
- 证券登记结算机构应妥善保存登记、存管、结算原始凭证及资料，保存期限不少于 20 年。
- 投资者委托证券公司交易，应通过证券公司申请证券账户。
- 证券登记结算机构作为中央对手方提供结算服务时，是共同清算对手方。

### 易错点

- 公司合并分立程序不要漏“债权人通知或公告”。
- 有限责任公司股东人数是 1-50 人。
- 90% 以上持股合并有特殊简化规则，但仍要通知其他股东。
- 禁止交易行为中“反向交易”常被做成场景题。

### 快速背诵句

法规第一章抓五条线：公司法、证券法、基金法、期货法、私募条例；证券法抓三公、披露、禁止交易、投资者保护。

## P10 第 2 章：证券经营机构管理规范，25 分

### 截图看到的重点

- 内部控制原则。
- 证券公司合规管理。
- 全面风险管理。
- 声誉风险管理。
- 适当性管理禁止行为。

### 必背点

- 内部控制原则包括健全、合理、制衡、独立。
- 内部控制应统一、全面、审慎，并贯彻有效性、及时性。
- 合规部门中具备 3 年以上证券、金融、法律、会计、信息技术等有关领域工作经历的合规管理人员数量不得低于公司总部人数的一定比例，且不得少于 5 人。
- 分支机构应配备合规管理人员。
- 投资银行、债券等合规风险较高的业务部门，应配备专职合规管理人员。
- 全面风险管理覆盖各类风险、各项业务、各个部门、分支机构和子公司。
- 声誉风险管理原则包括全程全员、预防第一、审慎管理、快速响应。
- 引发声誉风险的因素包括战略变动、业务产品、客户投诉、伤害传导、不实传言等。
- 适当性管理中，禁止向不符合准入要求的投资者销售产品或提供服务。
- 禁止向投资者作出不当判断，或告知投资者有可能使其误以为具有确定性的意见。

### 易错点

- 合规管理不是合规部门一个部门的职责，而是全员职责。
- 声誉风险不是“公关问题”，属于证券公司全面风险管理范围。
- 适当性管理不能只看客户是否签字，要看是否真正匹配和充分告知。

### 快速背诵句

经营机构管理抓五件事：治理、内控、合规、风控、适当性。

## P11 第 3 章：证券公司业务规范，25 分

### 截图看到的重点

- 证券经纪业务概念。
- 资产管理业务相关规范。
- 资产证券化业务。
- 中间介绍业务。
- 区域性股权市场业务。

### 必背点

- 证券经纪业务是证券公司开展证券交易营销、接受客户委托开立账户及处理交易指令、办理清算交收等经营性活动。
- 证券公司应当在公司登记机关登记证券经纪业务。
- 资产管理业务是证券公司受托管理客户资产。
- 资产管理业务中，现金类资产包括现金、银行存款、债券回购、货币市场基金等。
- 资产证券化业务中基础资产应符合规定，可以产生独立、可预测的现金流。
- 原始权益人、管理人、托管人各有职责，专项计划资产应独立。
- 证券公司从事中间介绍业务，应当建立客户资料和身份真实性审查制度，向客户充分揭示风险，并保存开户资料和影像资料。
- 期货公司与证券公司应签订书面委托协议，明确双方权利义务。
- 证券公司可以在区域性股权市场开展推荐企业挂牌、挂牌企业展示、承销可转换为股票的公司债券、推荐本公司承销的可转换债券进入区域性股权市场转让等业务。

### 易错点

- 经纪业务不是“帮客户决策”，只是代理客户交易。
- 资管业务不得承诺保本保收益。
- 中间介绍业务不是期货经纪业务本身，证券公司不能越界。
- 区域性股权市场业务范围容易多选漏选。

### 快速背诵句

业务规范逐个背红线：经纪不代客，投顾不下单，资管不保本，自营要隔离，融资融券看担保。

## P12 第 4 章：典型违法违规行为及法律责任，5 分

### 截图看到的重点

- 证券一级市场。
- 擅自公开或变相公开发行证券。
- 欺诈发行证券罪。
- 违规披露、不披露重要信息罪。
- 擅自改变公开发行证券募集资金用途。

### 必背点

- 擅自公开发行或变相公开发行证券包括非公开转让后股东人数超过 200 人等情形。
- 欺诈发行证券罪可涉及非法募集资金金额、直接损失、虚增资产或营业收入、虚增利润等标准。
- 违规披露、不披露重要信息罪主体包括依法负有信息披露义务的公司、企业及相关责任人员。
- 违规披露行为包括提供虚假财务会计报告，或者对依法应披露的重要信息不按规定披露。
- 改变公开发行证券募集资金用途，必须按招股说明书或其他公开发行募集文件所列资金用途使用，改变用途需经股东大会作出决议。

### 易错点

- 一级市场违法重点围绕“发行、披露、募集资金用途、中介证明文件”。
- 非公开发行一旦突破法定人数或公开化，可能变成变相公开发行。
- 募集资金用途不能随意改。

### 快速背诵句

一级市场违法看三件事：发行是否批准，披露是否真实，资金用途是否合规。

## P13 第 5 章：行业文化、职业道德与从业人员行为规范，10 分

### 截图看到的重点

- 从业人员条件。
- 董监高、分支机构负责人薪酬递延。
- 保荐代表人行为规范。
- 投资经理薪酬递延机制。

### 必背点

- 从业人员应具备条件，包括不存在特定违法违规情形、未被采取证券市场禁入措施或禁入期已届满等。
- 董监高、分支机构负责人薪酬递延应与合规风控、廉洁从业、履职尽责等挂钩。
- 保荐机构及其保荐代表人应遵守法律法规和业务规则，诚实守信、勤勉尽责，尽职推荐发行人证券发行上市，持续督导发行人规范运作。
- 投资经理参与跟投安排的人员薪酬与激励合计超过一定比例的，应适用薪酬递延要求，递延周期不少于 2 年。

### 易错点

- 行业文化不是纯记忆口号，常和廉洁从业、诚信管理、市场禁入、薪酬递延结合考。
- 保荐代表人不是只负责发行阶段，还涉及持续督导。
- 从业人员条件和禁止行为容易出判断题。

### 快速背诵句

从业人员最后抓四个词：诚信、廉洁、勤勉、合规。

# 第四部分 考前压缩背诵版

## 金融基础 30 句

- [ ] 直接融资是资金需求方通过发行股票、债券等工具直接向资金供给方融资。
- [ ] 间接融资通过银行等金融中介完成。
- [ ] 货币政策目标包括币值稳定、经济增长、充分就业、国际收支平衡。
- [ ] 科创板服务科技创新企业，创业板服务成长型创新创业企业，北交所服务创新型中小企业。
- [ ] 场内市场以交易所市场为代表，场外市场包括股转系统、柜台市场、区域股权市场。
- [ ] 投资者适当性是让适合的投资者购买适合的产品或服务。
- [ ] 证券业协会是行业自律组织。
- [ ] 证监会是行政监管机构。
- [ ] 股票是股权凭证。
- [ ] 债券是债权凭证。
- [ ] 普通股股东享有重大决策参与权、资产收益权、剩余财产分配权。
- [ ] 股票理论价格是未来收益的现值。
- [ ] 债券反映发行人与投资者之间的债权债务关系。
- [ ] 金融债券发行主体通常为金融机构。
- [ ] 资产证券化以基础资产现金流为支持。
- [ ] 中小非金融企业集合票据具有统一设计、统一冠名、统一增信、统一注册等特点。
- [ ] 基金管理人负责投资运作。
- [ ] 基金托管人负责资产保管和监督。
- [ ] 基金信息披露分募集信息披露和运作信息披露。
- [ ] 基金估值应重视公允性、一致性和公开性。
- [ ] 金融衍生工具价值依赖基础资产。
- [ ] 远期通常非标准化。
- [ ] 期货通常标准化并集中交易。
- [ ] 期权买方有权利无义务，卖方有履约义务。
- [ ] 互换是按约定交换现金流。
- [ ] 风险偏好是愿意承担的风险类型和水平。
- [ ] 风险容忍度是风险偏好的具体化边界。
- [ ] 风险限额用于把风险控制在可承受范围内。
- [ ] 压力测试检验极端情景下的承受能力。
- [ ] 金融第 3、4、5、6 章是本视频标注的高分核心。

## 法律法规 30 句

- [ ] 公司合并、分立要履行决议、编制清单、通知或公告债权人等程序。
- [ ] 有限责任公司由 1 个以上 50 个以下股东设立。
- [ ] 分公司不具有法人资格，子公司具有法人资格。
- [ ] 国有独资公司董事会成员中应有职工代表。
- [ ] 证券发行交易遵循公开、公平、公正原则。
- [ ] 禁止内幕交易、操纵市场、虚假陈述和欺诈客户。
- [ ] 证券登记结算资料保存不少于 20 年。
- [ ] 内部控制应健全、合理、制衡、独立。
- [ ] 合规管理是全员、全过程、全覆盖。
- [ ] 全面风险管理覆盖各类风险、各项业务、各部门和分支机构。
- [ ] 声誉风险管理强调预防第一、快速响应。
- [ ] 适当性管理要求了解客户、了解产品、风险揭示、适当匹配。
- [ ] 禁止向不符合准入要求的投资者销售产品或提供服务。
- [ ] 经纪业务是代理客户交易，不是代客户决策。
- [ ] 经纪业务不得接受全权委托。
- [ ] 投顾可以提供建议，不得替客户作出交易决定。
- [ ] 资管业务是受托管理客户资产。
- [ ] 资管业务不得承诺保本保收益。
- [ ] 资产证券化基础资产应能产生独立、可预测现金流。
- [ ] 中间介绍业务中证券公司不能越界从事期货经纪业务。
- [ ] 区域性股权市场属于场外市场。
- [ ] 擅自公开或变相公开发行证券属于一级市场违法重点。
- [ ] 欺诈发行证券重点看发行文件虚假和重大遗漏。
- [ ] 违规披露、不披露重要信息重点看披露义务主体和信息重大性。
- [ ] 募集资金用途不得擅自改变。
- [ ] 从业人员应诚实守信、勤勉尽责。
- [ ] 保荐代表人要尽职推荐和持续督导。
- [ ] 廉洁从业要求不得谋取或输送不正当利益。
- [ ] 诚信信息影响执业声誉和职业发展。
- [ ] 法规第 1、2、3 章是本视频标注的高分核心。
